# MERN Portfolio Web Application

## Project Overview

This is a **personal portfolio web application** built with the **MERN stack (MongoDB, Express.js, React.js, Node.js)**. It showcases projects, skills, and personal information in a clean, responsive interface. The app is designed to help users (potential employers or collaborators) easily explore my work and contact me.

**Live Website:** https://mern-portfolio-t58u-2epbxvi91-chloe-nitcheus-projects.vercel.app/

---

## Features

* Responsive design for desktop and mobile devices
* Interactive portfolio sections for projects, skills, and experience
* Smooth animations and user-friendly navigation
* Easily extendable and maintainable structure

---

## Technology Stack

* **Frontend:** React.js, Tailwind CSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Hosting:** Vercel (frontend), Render (backend)
* **Others:** Axios, Git for version control

---

## Installation and Running Locally

1. **Clone the repository**

```bash
git clone https://github.com/Chloe-ntc/MERN-PORTFOLIO
cd MERN-PORTFOLIO
```

2. **Install dependencies**

```bash
npm install
```

3. **Start the development server**

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser. The app will reload automatically if you make changes.

---

## Building for Production

To create an optimized production build:

```bash
npm run build
```

The output will be in the `build` folder, ready to be deployed to Vercel or another hosting platform.
